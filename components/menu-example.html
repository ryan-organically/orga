Universal Menu Component Example

This page demonstrates how to use the <universal-menu> Web Component.

How It Works:

  1. The component is defined once in components/universal-menu.js
  2. You load the script once per page (see below)
  3. You use the <universal-menu></universal-menu> tag anywhere
  4. The component automatically includes its own HTML, CSS, and JavaScript

What's a Web Component?

Web Components are a set of native browser APIs that let you create reusable custom HTML elements. Think of it like React components, but built into the browserâ€”no framework needed.

Benefits:

  • Zero dependencies - No React, Vue, or other frameworks needed
  • Reusable - Write once, use everywhere
  • Encapsulated - CSS and JS don't leak out
  • Native - Supported by all modern browsers
  • Lightweight - Just vanilla JavaScript
